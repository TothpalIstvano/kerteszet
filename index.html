    Objective:

    Create a form with a textbox and a button. The textbox can hold a number that will represent the number of rows and columns in a table. Once the submit button is clicked you will generate a two dimensional array with each of the elements of the array holding a random number between 1 and 100. After creating the array you will loop through it and the number is a multiple of 3, display the cell with a red background color; if not a multiple of 3 but a multiple of 2, display the cell with a blue background.

    Extras: Alert user if entry is not a number (NaN)
